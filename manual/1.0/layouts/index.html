The EximeeBPMS Manual

Welcome to the EximeeBPMS Manual! EximeeBPMS is a light-weight, open-source platform for Business Process Management. The Manual introduces key concepts in EximeeBPMS, provides installation procedures as well as a comprehensive reference section.

EximeeBPMS is an open-source fork of discontinued since 2025 Camunda 7 business process management platform.

Getting Started

The Guide Get started with EximeeBPMS is the quickest and easiest way to get up and running with EximeeBPMS. Alternatives include:

DMN 1.3
Spring Boot
